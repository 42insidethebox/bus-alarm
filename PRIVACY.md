# BusBell Privacy Policy

Last updated: 13 July 2026

BusBell does not create user accounts and does not collect, transmit, sell, or share personal data. Timetables, preferences, backups, and saved locations are stored locally on the user's device.

BusBell requests notification permission to deliver departure reminders. It requests foreground location permission when a user saves a place and optional background location permission to detect entry and exit for user-created places. Coordinates and geofence membership are processed on-device and are not transmitted to BusBell or third parties.

The optional place map has two modes. Offline geometry mode does not request map tiles. While the street-map view is open, MapLibre requests the visible map tiles from OpenFreeMap; those requests necessarily reveal the viewed map area and normal network metadata to that provider. BusBell sends no account identifier or analytics, and the user can switch to offline view at any time.

If the user photographs or chooses an image of a timetable, BusBell performs text recognition entirely on the device using the operating system or bundled recognition model. The image and raw recognized text are used only for the temporary import review and are not uploaded or retained in the timetable database. BusBell does not send timetable images to Hugging Face or any other AI service.

If the user imports a GTFS transit ZIP, BusBell reads, validates, indexes, and queries the feed locally. Nearby-stop searches compare the user's current or saved coordinates with stop coordinates on-device. BusBell does not upload the feed, route selection, or location search. The original transit data remains subject to its publisher's license and attribution terms.

Deleting the application removes its locally stored data. Users can create a portable JSON backup or CSV export from the Settings screen; the user controls where that copied data is stored or shared.

Questions about this policy can be directed to the support contact published with the App Store or Play Store listing.
